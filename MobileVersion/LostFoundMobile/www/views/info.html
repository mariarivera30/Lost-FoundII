About us

LostFound is a web-based system that will allow users to report lost or found items on Campus. The system provides a simple and user friendly interaction with the user. Every user that wants to browse across any of the versions of this web-app will be able to do so without any registration, in case of reporting an item or making a review, some requirements may be needed. LostFound consists of features that can make people reunite with their valuables through a very intuitive navigation across the web.
